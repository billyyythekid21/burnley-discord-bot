burnley.

This Discord bot was coded using Python with some databases utilising JSON. It started in the middle of 2024 as a small side-project for educational purposes, now it's transformed into a full-fledged multi-feature Discord application based off Object-Oriented Programming (OOP).




The Creator.

Billy Wu (billyyythekid21) - First-year Bachelor of Computer Science student, at https://billyyythekid21.github.io.




Commands.

Various features are available with burnley, for example chess between two Discord users, Wikipedia search, a music player, Google AI and more.
